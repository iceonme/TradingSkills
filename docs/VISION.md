## 🌟 一句话定位

**Trading Skills (TS) 是一套专为「AI 智能体 (Agent)」打造的全新量化交易工具包。** 
它将传统的单体量化脚本，重构为高度标准化、可灵活插拔的**“交易技能包(Skill)”**。无论 Agent 是否懂交易，只需如调用 EXE 般加载这些技能，即可瞬间具备专业的行情感知、策略研判与实盘执行能力。

---

## 📦 核心工具包模块 (The Four Pillars)

TS 抛弃了臃肿的单体脚本，将量化交易解构为四个独立且专业的 Skill 模块。Agent 可根据场景自由编排：

1. **🧠 策略计算模块 (Strategy Skills)**
   - **定位**: 纯粹无状态的“数学大脑”。
   - **功能**: 不含任何网络通信，仅接收标准数据流，输出确定的交易信号 (Signal)。包内自带可视化 Dashboard 的配置协议。
   - **特性**: 高度安全，回测与实盘环境的代码**100% 同一性**。

2. **📡 数据感知模块 (Data Feed Skills)**
   - **定位**: 连接真实世界的“眼睛”。
   - **功能**: 封装各大交易所 (OKX, Binance) 或链上节点的 API/WS 连接，清洗并输出标准化的实时或历史行情数据流。

3. **🦾 交易执行模块 (Execution Skills)**
   - **定位**: 经授权发单的“手足”。
   - **功能**: 接管具体的账户管理与下单动作，支持模拟盘 (Paper Trading) 与真实盘 (Live Trading) 的无缝切换。

4. **⚙️ 策略编排器模块 (Runner / Orchestrator)**
   - **定位**: 系统调度与时间流逝的“大管家”。
   - **功能**: 将【数据】->【策略】->【执行】串联为一条运转流水线，负责驱动整个闭环在真实时间轴或回测历史时间轴上平稳运行。

## 🏟️ 特色服务：Trading Agent Arena (交易竞技场)

除了基础构件，TS 还内置了一个极具特色的**“评价枢纽” —— Trading Agent Arena Skill**。

- **门槛极低**: 任何进入该系统的 Agent，不管自身有没有携带策略，只要加载了 Arena Skill，即可瞬间注册参赛。
- **实战检验**: Arena 将为参赛的 Agent 统一分配虚拟资金与标准化历史行情，通过严苛的回测沙箱考验其决策的胜率、回撤与夏普比率。
- **优胜劣汰**: 这不仅是一个回测工具，更是未来筛选、评级和定价顶级 AI 交易员的修罗场。

## 🔮 我们的愿景

在过去，AI 想要做交易，需要跨越复杂的接口调用、时间轴对齐、数据清洗等数不清的“工程泥沼”。

**今天，Trading Skills 要做的是铺设一条高速公路。** 我们提供“开箱即用”的内置技能，也制定“万物皆可接入”的开源协议。

我们的终极愿景是：**让每个聪明的 Agent 都能毫不费力地成为华尔街级的 Quant 猎手。**
